系统四大核心角色流程

普通申报人

在线自主填报课题、选择研究方向。可上传正式PDF/Word格式任务书及完全匿名的设计论证活页附件，实时接收审核与结果通知。

申报单位管理员

负责本机构（医院、高校、科研所）的员工资质审核，在系统核定名额上限内指派具体用户的课题申报授权，初审本单位申报课题并上报。

评审专家

系统自动进行单位回避及研究方向智能匹配指派。专家可在线接收或拒绝邀请，在完成双盲匿名评审及电子签名后，评审权限自动锁定防篡改。

省局超级管理员

掌管全局课题名额配置、申报周期发布；进行课题格式终审，智能推荐及指派专家；自动统计评审结果并发布最终官方立项公告，支持报表导出。

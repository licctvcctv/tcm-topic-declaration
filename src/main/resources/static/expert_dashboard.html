📋 立项评审原则（点击展开）

请根据以下维度对课题进行综合评分（满分100分）：

  1. 科学性与创新性（30分） — 研究选题是否具有明确的科学问题，研究思路是否具有创新性
  2. 研究方案可行性（25分） — 技术路线是否清晰，研究方法和指标是否合理可行
  3. 预期成果与应用价值（20分） — 预期成果是否明确，对中医药发展或临床实践是否有指导意义
  4. 研究基础与条件（15分） — 申报团队是否有相关研究积累，软硬件条件是否满足需求
  5. 经费预算合理性（10分） — 经费预算编制是否合理、合规

▸ 评分≥60分为合格；
▸ 立项推荐基于评分及综合判断，≥50%专家推荐立项方可进入审批流程。
